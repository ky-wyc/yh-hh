# QQ 群机器人一期开发状态

## 当前进度

一期目标是从 MVP 进入可长期运营阶段，范围以 `一期产品蓝图.md` 为准。

## 已完成

### 记忆最小版本，部分完成

- 已新增长期记忆数据表。
- 已支持 `/remember 内容` 创建待审核记忆。
- 已支持 `/forget 记忆ID` 软删除自己的记忆。
- 已支持后台新增、修改、审核、删除记忆。
- 已支持记忆来源、可信度、审核状态。
- AI 回复默认只读取 `approved` 记忆。
- `pending`、`rejected`、`deleted` 记忆不会参与 AI 回答。
- 记忆新增、修改、删除写入审计日志。

## 未完成

- 知识库最小版本。
- 每日总结和定时任务。
- 持久游戏状态。
- Skill 开关。
- 群管理详情。
- 基础群管增强：临时禁言、刷屏阈值、新人欢迎、手动解除禁言。
- 备份和恢复演练。

## 下一步建议

1. 完善记忆最小版本的边界：更多权限测试、敏感信息提示、后台筛选体验。
2. 开始知识库最小版本：后台录入纯文本、分块、群级隔离、`/kb` 问答入口。
3. 增加数据库迁移机制，减少生产环境表结构演进风险。
