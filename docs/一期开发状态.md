# QQ 群机器人一期开发状态

## 当前进度

一期目标是从 MVP 进入可长期运营阶段，范围以 `一期产品蓝图.md` 为准。

## 已完成

### 记忆最小版本，基础版本完成

- 已新增长期记忆数据表。
- 已支持 `/remember 内容` 创建待审核记忆。
- 已支持 `/forget 记忆ID` 软删除自己的记忆。
- 已支持后台新增、修改、审核、删除记忆。
- 已支持记忆来源、可信度、审核状态。
- AI 回复默认只读取 `approved` 记忆。
- `pending`、`rejected`、`deleted` 记忆不会参与 AI 回答。
- 记忆新增、修改、删除写入审计日志。

### 知识库最小版本，基础版本完成

- 已新增知识库文档表和分块表。
- 已支持后台录入纯文本知识。
- 已支持文本自动分块。
- 已支持确定性文本向量化并写入 `embedding_json`。
- PostgreSQL 环境中可用 pgvector 时会自动尝试创建扩展和 `embedding_vector` 列。
- 已支持后台配置知识库 Embedding：默认本地确定性向量，也可切换 OpenAI-compatible embedding API。
- 已支持 Embedding API Key 脱敏、清空和后台测试。
- 知识库重建索引会使用当前 Embedding 配置；外部向量化失败时会写入 `index_status=failed` 和 `index_error`。
- 当外部 Embedding 查询临时失败时，知识库检索会保留关键词分数兜底，避免直接中断查询。
- 已支持知识库作用域：全局或指定群。
- 已支持后台新增、修改、启停、删除知识库文档。
- 已支持后台检索测试。
- 已支持后台手动重建知识库索引。
- 已支持后台按群号和索引状态筛选知识库文档。
- 已支持后台批量重建当前筛选范围内的已启用文档索引。
- 已支持后台一键重试失败索引，并返回总数、成功数、失败数和跳过数。
- 已支持后台查看索引错误信息。
- 已支持 `/kb 问题` 查询知识库。
- `/kb` 返回来源标题和片段编号。
- 已验证 A 群知识库不会命中 B 群知识库。

### 工程化能力，部分完成

- 已增加轻量数据库迁移记录表 `schema_migrations`。
- 服务启动时会记录已执行迁移。
- 已支持对旧表进行幂等字段修补，为后续表结构演进预留路径。
- 已支持旧库自动补建一期新增表，云端拉取新版本后可随服务启动迁移。

### 定时任务和每日总结，基础版本完成

- 已新增定时任务表和任务执行历史表。
- 已支持后台新增、编辑、启停、删除定时任务。
- 已支持后台查看任务执行历史。
- 已支持一次性群提醒。
- 已支持基础每日群聊总结，基于近期消息日志生成消息数和最近片段。
- 已支持定时清理短期上下文缓存。
- 定时群提醒和每日总结会检查目标群是否启用，避免绕过后台群开关。

### 后台一期增强，部分完成

- 已支持全局 Skill 开关。
- 已支持群级 Skill 开关。
- 已支持后台群详情抽屉，展示消息、回复、记忆、知识库、关键词和定时任务统计。
- 已支持高风险 Skill 开关变更写入审计日志。
- 消息路由已接入 Skill 开关：AI、知识库、记忆、娱乐、基础群管等功能可被后台关闭。
- `/help` 会根据当前群实际启用的 Skill 展示可用命令。

### 持久游戏状态，基础版本完成

- 已新增游戏状态表 `game_states`。
- 已支持 `/guess start` 启动猜数字游戏。
- 已支持 `/guess 数字` 参与猜数字游戏。
- 已支持 `/guess stop` 结束猜数字游戏。
- 游戏状态按群隔离。
- 同一群同一时间只允许一个活跃猜数字游戏。
- 未过期游戏状态落库保存，服务重启后可继续。
- 过期游戏状态可由定时清理任务标记为 `expired`。
- 猜数字命令不调用大模型。
- 后台群详情已展示活跃游戏数量。

### 基础群管增强，基础版本完成

- 已支持 `/mute @用户 秒数 原因` 临时禁言。
- 已支持 `/unmute @用户 原因` 手动解除禁言。
- 已支持后台群详情中配置新人欢迎开关和欢迎语。
- 已支持 OneBot `group_increase` 事件触发新人欢迎。
- 已支持后台群详情中配置刷屏检测阈值、窗口时间和禁言秒数。
- 已支持后台配置违规累计窗口、阶梯处罚开关、处罚倍率和禁言封顶。
- 已支持普通用户刷屏时自动临时禁言。
- 已支持基于近期 `warn`、`mute`、`flood_mute` 审计记录累计用户违规次数。
- 已支持刷屏重复违规按累计次数自动提升禁言时长，并保留禁言封顶。
- 已支持后台群详情展示近期违规用户、累计次数和最近时间。
- `/warn` 可记录目标用户近期累计违规次数。
- 群管命令、自动刷屏禁言、新人欢迎均写入审计日志。
- 普通用户不能执行禁言、解禁等管理员命令。

### 备份和恢复演练，工具与记录模板完成

- 已支持 `scripts/backup.sh` 生成 PostgreSQL SQL 备份。
- 备份文件生成后会检查非空，并尽量设置为 `600` 权限。
- 已支持 `scripts/restore_postgres.sh` 从 SQL 备份恢复 PostgreSQL。
- 已新增 `scripts/backup_restore_drill.py` 生成非破坏性备份恢复演练报告。
- 演练报告会检查 `.env`、备份文件、PostgreSQL 标识符、NapCat 登录态目录和恢复命令。
- 演练报告明确标记 `restore_executed=false`，避免把准备检查误认为真实恢复。
- 运维文档已补充演练步骤和证据留存清单。

### 一期完成定义审计，工具与清单完成

- 已新增 `scripts/phase_one_audit.py`，用于静态检查一期代码、测试、后台页面、运维脚本、文档和敏感文件跟踪状态。
- 审计脚本不连接 QQ、不连接数据库、不读取 `.env`、不执行恢复，适合本地、CI 和云端重复运行。
- 审计报告会把“真实群持续运行一周”和“真实恢复演练证据”标记为 `manual_required`，避免把本地代码证据误判为完整上线完成。
- 已新增 `docs/一期完成定义证据清单.md`，把一期完成定义拆成本地自动证据、云端真实证据和不能由脚本替代的内容。
